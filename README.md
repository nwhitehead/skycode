# SkyCode

> [!WARNING]
> In development, not done!

SkyCode is an open source AI coding agent.

This application is an open source coding agent harness inspired by
[ClaudeCode](https://github.com/anthropics/claude-code),
[OpenCode](https://github.com/anomalyco/opencode),
and [OpenClaudia](https://github.com/dollspace-gay/OpenClaudia).

Features:

* 100% open source
* Compatible with multiple model providers
    * State-of-the-art GLM open source models with [Z.ai](https://z.ai/chat)
    * Local LLM server with [ollama](https://github.com/ollama/ollama)
    * Anthropic, OpenAI, Google Gemini etc.
* Quick select agent roles and permissions
* Parallel agent tracking and supervision
* Persona support
* Deep agentic tools support
* Stateful memory
* Task tracking
* Automatic context compaction and optimization
